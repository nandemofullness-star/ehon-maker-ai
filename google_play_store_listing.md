## 2. ストアリスト（日本語）

### アプリ名（30文字以内）
```
絵本メーカーAI
```

### 短い説明（80文字以内）
```
写真をAIで絵本風イラストに変換。KDP出版用PDFを簡単作成。
```

### 詳細説明（4000文字以内）
```
■ 絵本メーカーAI とは？

スマートフォンの写真を使って、Kindle Direct Publishing（KDP）向けの絵本PDFを簡単に作成できるアプリです。Google Gemini AIが写真を温かみのある絵本風イラストに自動変換します。

■ 主な機能

【写真のアップロード・管理】
・スマートフォンのギャラリーから複数の写真を一度に選択できます
・ページの順番を自由に並び替えられます
・不要なページは簡単に削除できます

【AIによる絵本風変換】
・Google Gemini AIが写真を3歳向けの可愛い絵本イラストに変換
・太いアウトライン、明るいパステルカラー、温かみのあるフラットイラストスタイル
・一括変換機能で全ページを自動処理

【テキスト追加】
・各ページに物語のテキストを入力できます
・テキストはPDF出力時に画像に合成されます

【KDP用PDF出力】
・Amazon KDPペーパーバックの標準サイズ（215.9×215.9mm、300DPI）に対応
・高品質なPDFファイルを生成
・生成したPDFはシェア機能でGoogleドライブ、メール等に保存可能

■ KDP（Kindle Direct Publishing）とは？

Amazonが提供するセルフ出版サービスです。このアプリで作成したPDFをKDPにアップロードすることで、世界中に向けて自分だけの絵本を出版できます。

■ こんな方におすすめ

・子供のために手作り絵本を作りたい親御さん
・旅行の思い出を絵本にしたい方
・ペットの写真で絵本を作りたい方
・KDPで絵本を出版したいクリエイター

■ 注意事項

・AI変換機能はインターネット接続が必要です
・KDPペーパーバックには最低24ページが必要です
・生成されたPDFはKDPの仕様に合わせて最適化されています

■ プライバシー

アップロードされた写真はAI変換処理のみに使用され、サーバーに保存されません。
```

## 3. スクリーンショット要件

Google Playでは以下のスクリーンショットが必要です：

| 種類 | サイズ | 枚数 |
|------|--------|------|

| タブレット10インチ（任意） | 最小1200×1920px | 0〜8枚 |

### 推奨スクリーンショット内容

1. **ホーム画面（空状態）** - アプリの第一印象

3. **AI変換中の画面** - AI機能のデモ
4. **AI変換完了後の比較** - 変換前後の違い
5. **PDF出力・共有画面** - 最終成果物

## 8. Google Play Developer Console での公開手順

### 事前準備
1. [Google Play Console](https://play.google.com/console) にアクセス
2. デベロッパーアカウントを作成（$25 一回払い）
3. 本人確認を完了

### アプリ公開手順

**ステップ1: 新しいアプリを作成**
- 「アプリを作成」をクリック
- アプリ名: 「絵本メーカーAI」
- デフォルト言語: 日本語
- アプリの種類: アプリ
- 無料/有料: 無料

**ステップ2: ストアの掲載情報を設定**
- 上記の説明文・スクリーンショット・アイコンをアップロード
- フィーチャーグラフィックをアップロード

**ステップ3: コンテンツレーティングを設定**
- 上記のアンケート回答を参考に設定

**ステップ4: ターゲット国・価格を設定**
- 対象国: 日本（または全世界）
- 価格: 無料

**ステップ5: AABファイルをアップロード**
- Expo EASでビルドした `.aab` ファイルをアップロード
- 「本番」トラックを選択

**ステップ6: 審査・公開**
- 「審査のために送信」をクリック
- 通常1〜3日で審査完了

## 9. EAS Build コマンド（AABファイル生成）

アプリをGoogle Playに公開するには、AABファイルが必要です。
Expo EAS Buildを使用してビルドします：

```bash
# EAS CLIのインストール
npm install -g eas-cli

# EASにログイン
eas login

# eas.json の設定（プロジェクトルートに作成）
# {
#   "build": {
#     "production": {
#       "android": {
#         "buildType": "app-bundle"
#       }
#     }
#   }
# }

# Androidビルドの実行
eas build --platform android --profile production
```

ビルド完了後、EASダッシュボードからAABファイルをダウンロードできます。

## 10. キーストア（署名鍵）について

- EAS Buildを使用する場合、Expoが自動的にキーストアを管理します
- 独自のキーストアを使用する場合は `eas.json` で設定が必要です
- **重要**: キーストアは安全な場所に保管してください。紛失するとアプリの更新ができなくなります。
