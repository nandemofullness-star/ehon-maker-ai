# KDP絵本メーカーAI - デザイン計画

## アプリ概要
写真をアップロードしてGemini AIで絵本風イラストに変換し、テキストを追加してKDP（Kindle Direct Publishing）向けPDFを生成するAndroidアプリ。

---

## スクリーン一覧

| スクリーン | 説明 |
|-----------|------|
| ホーム（メイン） | 写真アップロード・ページ管理・AI変換・PDF出力の全機能を1画面に集約 |
| ページ詳細編集 | 個別ページのテキスト編集・AI変換・プレビュー |
| 設定 | Gemini APIキー設定 |

---

## 主要コンテンツと機能

### ホーム画面
- **ヘッダー**: アプリロゴ + タイトル「KDP絵本メーカーAI」
- **アップロードゾーン**: タップで複数画像選択（expo-image-picker）
- **コントロールバー**: ページ数表示 / 「AIで一括変換」ボタン / 「PDF出力」ボタン
- **ページグリッド**: 2列グリッド表示、各カードに画像プレビュー・ページ番号・テキスト・並び替えボタン・削除ボタン
- **ページ数警告**: 24ページ未満の場合に警告バナー表示
- **空状態**: アイコン + 案内テキスト

### ページカード（グリッドアイテム）
- 正方形プレビュー画像
- ページ番号バッジ（左上）
- AI変換済みバッジ（右上、緑チェック）
- 処理中オーバーレイ（スピナー）
- テキスト入力フィールド（下部）
- 上下移動ボタン + 削除ボタン

### 設定画面
- Gemini APIキー入力・保存（AsyncStorage）

---

## キーユーザーフロー

### 絵本作成フロー
1. ホーム画面でアップロードゾーンをタップ
2. 端末ギャラリーから複数写真を選択
3. ページグリッドに写真が追加される
4. 各ページのテキスト入力フィールドに文章を入力
5. 必要に応じてページを上下に並び替え
6. 「AIで一括変換」ボタンをタップ → サーバーAPIで絵本風イラストに変換
7. 変換完了後、「PDF出力」ボタンをタップ
8. PDFが生成され、シェアシートで保存・共有

## デザイン原則

- **シンプルさ**: 1画面で全操作完結、タブは最小限
- **視覚的フィードバック**: AI変換中はスピナー + ブラー効果
- **モバイルファースト**: 縦スクロール、大きなタップターゲット
- **日本語対応**: フォントはシステムフォント（Hiragino/Noto）を使用
